# Double Slash Podcast Website

WIP...

## Quick Start

We use [pnpm](https://pnpm.io) on package management.

#### Install dependencies

```
pnpm install
```

#### Run dev mode

```
pnpm dev
```


#### Build application

```
pnpm build
```
