## Étude IA sur long terme

Alibaba a testé 18 agents de codage IA sur 100 bases de code réelles, sur une période de 233 jours pour chacune. Leur échec a été retentissant.
Réussir les tests une seule fois s'avère facile. Maintenir du code pendant 8 mois sans tout casser, c'est là que l'IA s'effondre complètement.
SWE-CI est le premier benchmark qui mesure la maintenance du code à long terme plutôt que les corrections de bugs ponctuelles. Chaque tâche suit 71 commits consécutifs d'évolution réelle.
75 % des modèles cassent du code fonctionnel lors de la maintenance. Seuls Claude Opus 4.5 et 4.6 affichent un taux de régression nul supérieur à 50 %. Tous les autres modèles accumulent une dette technique qui s'aggrave à chaque itération.
