## ArrowJS

Arrow requires no build step, no JSX compilation, no React compiler, no Vite plugin (there is one if you need SSR), no Vue template complier, and yet it runs incredibly fast at less than 5kb over the wire. When coupled with the [Arrow sandbox](https://arrow-js.com/#sandbox), it's perfect for interfaces produced by chat agents too.

- [https://arrow-js.com](https://arrow-js.com/)
